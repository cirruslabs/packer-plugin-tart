# Tart

Type: `tart`

The tart builder is used to create macOS VMs for Apple Silicon.


<!-- Builder Configuration Fields -->

### Required

- `vm_name` (string) - The name of the VM to be created.
- `vm_base_name` (string) - The name of the VM to be used for the initial cloning.

### Example Usage

Here is a basic example:

```json
{
  "type": "tart-local",
  "vm_name": "my-packer",
  "vm_base_name": "latest",
  "ssh_username": "admin",
  "ssh_password": "admin",
  "headless": "true",
}
```
